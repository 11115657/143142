# Blender 导出指南

## 角色

导出路径：`public/assets/characters/player/player.glb` 或 `public/assets/characters/enemies/grunt.glb`

需要动画：Idle、Walk、Run、Aim、Fire、Reload、Hit、Death。

## 武器

导出路径：`public/assets/weapons/rifle_a.glb`、`public/assets/weapons/pistol_a.glb`

建议武器朝向 -Z，枪口在模型前方，比例与第一人称相机协调。

## 地图

视觉模型：`public/assets/maps/training_base.glb`
碰撞模型：`public/assets/maps/collision_training_base.glb`
